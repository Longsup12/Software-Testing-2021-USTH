# Write a Java method with the signature
1.
```java 
import java.util.Vector;

class Vector { 
	public static Vector union(Vector x, Vector y) { 
		Vector v = new Vector(); 
		v.addAll(x); 
		v.addAll(y); 
		return v; 
	} 
	public static void main(String[] arg) { 
		Vector x = new Vector(); 
		x.add(1); 
		Vector y = new Vector(); 
		y.add(2); 
		Vector v = union(x, y); 
	} 
} 
```
